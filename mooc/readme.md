# mooc的资料结构
- 一般来说课程都是有相关主页的
- 相关的video都在各自的网站上,一般来说都在我的收藏夹中
- 课程的assignment我会直接放在mooc文件夹中
- 相关的project我会放在computerScience这个文件夹中
- 对相关的readings所作的笔记，我会放在notes文件夹中
- 对slides所作的笔迹放在平板上了
- 以xxxx课程(eg: 操作系统课程/公开课)的格式在类似知乎的平台上搜索会有相应的结果
